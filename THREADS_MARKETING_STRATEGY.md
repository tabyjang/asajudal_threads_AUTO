# Threads 사주분석 사이트 홍보 전략

## 핵심 원칙

1. **가치 제공 먼저, 홍보는 자연스럽게**
2. **일관된 포스팅 시간 (알고리즘 활용)**
3. **참여 유도 (댓글, 공유)**
4. **스팸으로 보이지 않기**

---

## 콘텐츠 전략 (5가지 유형)

### 1. 일일 운세 시리즈 (매일 아침 7-8시)
```
🌅 [12월 24일] 오늘의 띠별 운세

🐭 쥐띠: 오늘은 새로운 인연을 만날 수 있는 날!
💼 업무운 ★★★★☆
💰 재물운 ★★★☆☆

🐮 소띠: 차분하게 계획을 세우기 좋은 날
💼 업무운 ★★★★★
💰 재물운 ★★★★☆

...

👉 내 사주 상세 분석 받기: [사이트URL]
```

**효과**: 매일 확인하는 습관 형성 → 팔로워 증가

---

### 2. 미니 사주 팁 (매일 오후 1-2시)
```
💡 사주 미니 지식

"갑목(甲木)일주 특징"

🌳 큰 나무처럼 꿋꿋하고 정직한 성격
📈 리더십이 강하고 책임감이 뛰어남
💪 한 번 시작하면 끝까지 해내는 추진력

당신은 어떤 일주인가요?
👉 무료로 확인하기: [사이트URL]

#사주 #사주팔자 #일주 #운세
```

**효과**: 정보 가치 제공 → 신뢰도 증가

---

### 3. 사주 퀴즈/질문 (매일 저녁 7-8시)
```
🎯 사주 퀴즈!

다음 중 재물운이 가장 강한 일주는?

A) 갑목(甲木)
B) 경금(庚金)
C) 임수(壬水)
D) 무토(戊土)

💬 댓글로 답변 주세요!
정답은 다음 게시물에서 공개 👀

#사주퀴즈 #운세 #재물운
```

**효과**: 참여 유도 → 알고리즘 노출 증가

---

### 4. 실제 사례 (주 2-3회)
```
📖 실제 상담 사례 (개인정보 비식별)

"30대 직장인 A씨의 이직 고민"

A씨는 이직을 고민 중이었습니다.
사주를 보니 올해가 '식상운'이 강한 해...

✅ 결과: 이직 후 3개월 만에 승진!
💬 "사주 상담 받길 잘했어요" - A씨

당신도 고민이 있다면?
👉 [사이트URL]

#사주상담 #이직 #커리어
```

**효과**: 신뢰도 구축 → 실제 상담 전환

---

### 5. 계절/이벤트 특집 (주말)
```
🎄 크리스마스 특집!

"연말에 운이 풀리는 띠 TOP 3"

🥇 1위: 토끼띠 - 인연운 최고조
🥈 2위: 돼지띠 - 재물운 상승
🥉 3위: 양띠 - 사업운 좋음

당신의 연말 운세는?
👉 무료 사주 풀이: [사이트URL]

#연말운세 #크리스마스 #신년운세
```

**효과**: 시즌 관심사 활용 → 유입 폭증

## 콘텐츠 제작 팁

### 1. 이미지 디자인
- **Canva** 사용 (무료)
- 템플릿 활용:
  - 크기: 1080x1080 (정사각형)
  - 스타일: 심플, 가독성 좋은 폰트
  - 색상: 차분한 톤 (보라, 남색, 금색)

### 2. 텍스트 작성 원칙
- 첫 줄에 이모지 + 핵심 메시지
- 3-5줄로 간결하게
- 마지막에 CTA (Call To Action)
- 해시태그 3-5개

### 3. 사이트 링크 전략
- 직접 링크는 알고리즘에 불리할 수 있음
- **프로필에 링크 고정**
- 게시물에서는 "프로필 링크 확인" 또는 "DM 주세요"

## 자동화 시 주의사항

### ⚠️ 스팸 방지
1. **하루 최대 5-7개 게시물**
2. **같은 내용 반복 금지**
3. **팔로우/언팔 자동화 금지**
4. **자연스러운 시간 간격** (최소 2-3시간)

### ✅ 효과적인 자동화
1. **예약 포스팅만 자동화**
2. **댓글은 직접 답변** (자동화 X)
3. **주 1-2회는 실시간 라이브 포스팅**
4. **팔로워 DM은 직접 응대**

---

## 성장 전략 (단계별)

### 1단계: 기반 구축 (1-2개월)
- 목표: 팔로워 100-500명
- 방법:
  - 일관된 포스팅 (하루 3-5개)
  - 관련 해시태그 적극 활용
  - 다른 사주/운세 계정에 댓글 활동

### 2단계: 참여 증대 (3-4개월)
- 목표: 팔로워 500-2000명
- 방법:
  - 퀴즈/질문 콘텐츠 증가
  - 팔로워 사주 무료 풀이 이벤트
  - 다른 계정과 협업

### 3단계: 전환 최적화 (5-6개월)
- 목표: 실제 유료 상담 전환
- 방법:
  - 무료 기초 분석 제공
  - 상세 분석은 유료 유도
  - 후기 콘텐츠 제작

## 측정 지표

### 추적할 KPI
1. **팔로워 증가율** (주간)
2. **게시물 도달률** (조회수)
3. **참여율** (좋아요 + 댓글 / 조회수)
4. **사이트 클릭률** (UTM 파라미터 사용)
5. **실제 상담 전환율**

### 분석 도구

- Google Analytics (사이트 유입 추적)
- 스프레드시트로 수동 기록

## 법적/윤리적 주의사항

1. **의료/법률 조언 금지**
   - "질병이 낫는다" 등 의료 효과 주장 X
   - "소송에서 이긴다" 등 법률 보장 X

2. **과장 광고 금지**
   - "100% 적중" 같은 표현 자제
   - "참고용으로만" 명시

3. **개인정보 보호**
   - 상담 사례는 철저히 익명화
   - 동의 없이 실명/사진 사용 금지

## 결론

**핵심은 "가치 제공"입니다.**

단순히 사이트 링크만 올리는 게 아니라:
1. 매일 유익한 정보 제공
2. 팔로워와 소통
3. 자연스럽게 신뢰 구축
4. 그 결과로 사이트 방문 유도

일관성 있게 3-6개월 운영하면 확실한 효과를 볼 수 있습니다!
