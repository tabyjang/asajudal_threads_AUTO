# 띠별 운세 작성 가이드

## 솔직한 이야기

**띠별 운세는 엄밀히 말하면 "엔터테인먼트"에 가깝습니다.**

하지만 전통 사주명리학의 원리를 바탕으로 작성하면 일관성과 신뢰도를 높일 수 있습니다.

## 2. 일일 운세 작성 방법

### A. 오늘의 일진(日辰) 확인

**예시**: 2025년 12월 24일 = 갑자일(甲子日)
- 천간: 갑(甲) - 목(木)
- 지지: 자(子) - 수(水)

### B. 각 띠와의 관계 분석

오행 상생상극 원리 활용:
- **상생**: 수생목, 목생화, 화생토, 토생금, 금생수
- **상극**: 수극화, 화극금, 금극목, 목극토, 토극수

#### 예시: 갑자일(목+수)일 때

**좋은 띠**:
- 쥐띠(수): 일진과 같은 오행 - 편안한 날
- 호랑이/토끼띠(목): 수생목으로 힘을 받음 - 좋은 날
- 뱀/말띠(화): 목생화로 활력 - 활동적인 날

**보통인 띠**:
- 원숭이/닭띠(금): 금극목이지만 약한 영향
- 소/용/양/개띠(토): 목극토이지만 중립적

**조심해야 할 띠**:
- 돼지띠(수): 수가 너무 많아 균형 깨짐

## 3. 실전 작성 템플릿

### 템플릿 1: 간단 버전

```
🐭 [띠이름]: [한 줄 요약]
💼 업무운 [별점] 💰 재물운 [별점] ❤️ 연애운 [별점]
```

**예시**:
```
🐭 쥐띠: 오늘은 새로운 시작에 좋은 날! 미뤘던 일을 시작하세요.
💼 업무운 ★★★★☆ 💰 재물운 ★★★☆☆ ❤️ 연애운 ★★★★☆
```

```
🐭 [띠이름] ([출생연도])

📊 오늘의 키워드: [키워드 3개]

💼 업무/학업
[2-3줄 설명]

💰 재물/금전
[1-2줄 설명]

❤️ 연애/인간관계
[1-2줄 설명]

⚠️ 주의사항
[1줄]

💡 행운 TIP
- 행운 색상: [색]
- 행운 방향: [방향]
- 행운 숫자: [숫자]
```

## 4. 실용적인 작성 방법 (추천)

### 옵션 A: 일진 기반 자동 생성

**만세력/일진 API**를 사용하여 매일 자동으로 띠별 운세 생성

1. 만세력 API로 오늘의 일진 확인
2. 오행 상생상극 계산
3. 템플릿에 자동 대입

### 옵션 B: 일주일 단위 사전 작성

**월요일에 한 주치 작성**:
- 월요일: 새 시작, 계획 세우기
- 화요일: 실행력, 추진력
- 수요일: 중간 점검, 휴식
- 목요일: 마무리 준비
- 금요일: 성과 확인, 주말 준비
- 주말: 휴식, 재충전

각 요일 테마에 맞춰 12띠 운세 작성

### 옵션 C: 랜덤 + 밸런스

**완전히 랜덤하되, 밸런스 유지**:
- 매일 3-4개 띠는 "좋음"
- 3-4개 띠는 "보통"
- 3-4개 띠는 "주의"

## 5. 별점 부여 기준

### 업무운
- ★★★★★: 중요한 계약/프레젠테이션 성공
- ★★★★☆: 업무가 술술 풀리는 날
- ★★★☆☆: 평범한 날
- ★★☆☆☆: 실수 주의, 신중하게
- ★☆☆☆☆: 무리하지 말고 방어적으로

### 재물운
- ★★★★★: 예상치 못한 수입, 투자 성공
- ★★★★☆: 작은 이득, 절약 성공
- ★★★☆☆: 수입/지출 균형
- ★★☆☆☆: 지출 주의
- ★☆☆☆☆: 큰 지출 피하기

### 연애운
- ★★★★★: 고백/프러포즈 성공률 높음
- ★★★★☆: 좋은 만남, 달달한 데이트
- ★★★☆☆: 평범한 하루
- ★★☆☆☆: 오해 생길 수 있음
- ★☆☆☆☆: 싸움 주의

## 7. 월별/계절별 특징 반영

### 1월 (新年)
- 새해 목표, 새 시작, 계획 수립

### 2월 (설날)
- 가족, 전통, 귀향, 화합

### 3-5월 (봄)
- 성장, 발전, 새싹, 활력

### 6-8월 (여름)
- 에너지, 활동, 휴가, 여행

### 9-11월 (가을)
- 수확, 결실, 정리, 마무리

### 12월 (겨울)
- 휴식, 재충전, 연말 정산, 송년

## 8. 실제 작성 예시

### 2025년 12월 24일 (크리스마스)

#### 일진 확인
- 갑자일(甲子) - 목+수

#### 작성 과정

**1단계: 오행 관계 정리**
- 좋음: 쥐(수), 호랑이/토끼(목), 뱀/말(화)
- 보통: 원숭이/닭(금), 소/용/양/개(토)
- 주의: 돼지(수)

**2단계: 크리스마스 테마 반영**
- 좋음 → "특별한 선물", "좋은 만남"
- 보통 → "평화로운 하루", "여유"
- 주의 → "무리하지 말기", "휴식"

**3단계: 최종 작성**

```
🎄 [12월 24일 크리스마스] 띠별 운세

🐭 쥐띠: ❤️ 특별한 선물을 받을 수 있는 날! 기대하세요.
💼 ★★★★☆ 💰 ★★★★☆ ❤️ ★★★★★
```

**구글 시트/엑셀로 한 달치 미리 작성**:

| 날짜 | 일진 | 쥐 | 소 | 호랑이 | ... |
|------|------|----|----|--------|-----|

```
2025년 12월 24일은 갑자일(甲子, 목+수)입니다.
```

## 결론

**띠별 운세는 "재미 + 가벼운 조언"입니다.**

완벽한 명리학적 근거보다는:
1. **일관성** (같은 원리로 계속 작성)
2. **긍정성** (대부분 긍정적으로)
3. **실용성** (실제 도움이 되는 조언)

이 세 가지만 지키면 됩니다!
